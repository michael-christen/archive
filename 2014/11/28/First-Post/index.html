First Post

This is my very first blog post using Hexo, and
hosted on github. It details using the STM32F1 / STM32F4 processors as
I2C slaves, and modifying the OpenPilot source code to be used as an
I2C slave flight controller.

Slave transactions

I2C has 4 operating modes:

  • Slave receiver
  • Slave transmitter
  • Master transmitter
  • Master receiver
